# Reel Workspace - AI-Powered Instagram Reel Processing

A MERN + TypeScript application that processes Instagram Reels using AI for transcription, summarization, and OCR.

## Project Structure

```
/reel-workspace
├── /server          # Backend API (Node.js + Express + TypeScript)
├── /client          # Frontend (React + TypeScript) - Coming soon
└── README.md
```

## Tech Stack

- **Backend**: Node.js, Express, TypeScript, MongoDB
- **AI Services**: Google Gemini API
- **Media Processing**: Cloudinary, Cobalt API
- **Authentication**: JWT
