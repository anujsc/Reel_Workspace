# Utils

Helper functions.

This directory will contain utility functions and helpers used throughout the application.

## Structure

Utilities will include:

- `logger.ts` - Logging utilities
- `validators.ts` - Custom validation functions
- `formatters.ts` - Data formatting helpers
- `errors.ts` - Custom error classes
- `constants.ts` - Application constants
- `asyncHandler.ts` - Async error wrapper

Each utility will:

1. Be pure and reusable
2. Have a single responsibility
3. Be well-documented
4. Include type definitions
